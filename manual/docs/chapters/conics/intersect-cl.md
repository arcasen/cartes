### 直线与圆锥曲线的交点

#### 求解交点的参数化方法

设齐次坐标点与直线：

- **点**：平面上的点 $(x, y)$ 表示为向量 $\bm{x} = [x, y, w]^T$。
- **直线**：直线 $ax + by + c = 0$ 表示为向量 $\bm{l} = [a, b, c]^T$。

点在直线上的充要条件是：$\bm{l}^T \bm{x} = 0$。

点 $\bm{x}$ 在曲线上的充要条件是：**$\bm{x}^T \bm{C} \bm{x} = 0$**。

求直线 $\bm{l}$ 与曲线 $\bm{C}$ 的交点，本质上是在寻找满足 $\bm{l}^T \bm{x} = 0$ 且 $\bm{x}^T \bm{C} \bm{x} = 0$ 的向量 $\bm{x}$。

**第一步：参数化直线**

找到直线上任意两个不同的点 $\bm{u}$ 和 $\bm{v}$（即满足 $\bm{l}^T \bm{u} = 0$ 和 $\bm{l}^T \bm{v} = 0$）。

直线上任何一点都可以表示为它们的线性组合：

$$\bm{x} = \lambda \bm{u} + \mu \bm{v}$$

**第二步：代入二次型**

将参数化的 $\bm{x}$ 代入曲线方程：

$$(\lambda \bm{u} + \mu \bm{v})^T \bm{C} (\lambda \bm{u} + \mu \bm{v}) = 0$$

展开得到关于 $\lambda$ 和 $\mu$ 的齐次二次方程：

$$\lambda^2 (\bm{u}^T \bm{C} \bm{u}) + 2\lambda\mu (\bm{u}^T \bm{C} \bm{v}) + \mu^2 (\bm{v}^T \bm{C} \bm{v}) = 0$$

**第三步：求解比例 $\lambda / \mu$**

令 $\rho = \lambda/\mu$，上述方程是一个典型的一元二次方程。
* 通过求根公式解出 $\rho_1, \rho_2$。
* 将解代回 $\bm{x} = \rho \bm{u} + \bm{v}$，即可得到两个交点的齐次坐标。

- **你的公式**：$\bm{x} = \lambda \bm{u} + (1-\lambda) \bm{v}$
  这个公式隐含了一个约束：系数之和必须为 1。

- **通用公式**：$\bm{x} = \lambda \bm{u} + \mu \bm{v}$
  这里 $\lambda$ 和 $\mu$ 是独立的。如果解出来 $\mu = 0$，那么 $\bm{x} = \lambda \bm{u}$。如果 $\bm{u}$ 是无穷远点，我们就完美地捕捉到了这个交点。

1. 数学形式的对称性（齐次方程更好解）

使用 $\lambda \bm{u} + \mu \bm{v}$ 代入二次曲线方程 $\bm{x}^T C \bm{x} = 0$ 后，我们会得到一个非常漂亮的**二元二次齐次方程**：

$$A \lambda^2 + 2B \lambda\mu + D \mu^2 = 0$$

这个方程有一个巨大的优势：**它关于 $\lambda$ 和 $\mu$ 是对称的。**
这意味着我们可以根据 $A$ 和 $D$ 哪个更大，灵活地决定是“除以 $\mu^2$ 设 $\mu=1$”还是“除以 $\lambda^2$ 设 $\lambda=1$”。这种灵活性对于**数值稳定性**至关重要。

而如果你使用 $\lambda \bm{u} + (1-\lambda) \bm{v}$，代入后会得到：
$$(\bm{u} - \bm{v})^T C (\bm{u} - \bm{v}) \lambda^2 + \dots = 0$$
这实际上强制你选取了 $\bm{u} - \bm{v}$ 作为基向量。如果 $\bm{u}$ 和 $\bm{v}$ 选得不好（比如它们非常接近，或者它们的差向量导致数值很小），计算精度就会下降。

3. 几何意义的区别

- **$\lambda \bm{u} + (1-\lambda) \bm{v}$**：这是**仿射几何**的写法。它把直线看作是连接两点的线段及其延伸。它依赖于原点的选择（因为系数和必须为1）。
- **$\lambda \bm{u} + \mu \bm{v}$**：这是**射影几何**的写法。它把直线看作是由两个基向量张成的二维子空间（在三维齐次空间中）。它不依赖于具体的参数化方式，更加本质。

结论：虽然在欧几里得空间（普通 $x, y$ 坐标）里，你的写法完全没问题，但在处理矩阵形式的二次曲线（通常涉及齐次坐标）时：

**$\lambda \bm{u} + \mu \bm{v}$ 是更“高级”的写法，因为它能处理无穷远点，并且在编程时能通过选择 $\lambda=1$ 或 $\mu=1$ 来避免除以零的错误。**

#### Richter-Gebert 算法

Jürgen Richter-Gebert 算法[@RG2011, 194]的核心是构造一个特定的**秩 2 矩阵**，并将其分解为两个**秩 1 矩阵**（代表两个交点）。这个方法直接利用齐次坐标的代数性质。

<!--

## 核心算法原理：基于退化二次曲线的求解

在Jürgen Richter-Gebert的《Perspectives on Projective Geometry》中，求圆锥曲线（二次曲线）与直线的交点并不是直接解二次方程，而是采用一个更优雅的**代数几何**方法——利用**退化二次曲线**（degenerate conic）来求解。

该算法的核心思想是：将"求直线与圆锥曲线交点"问题转化为"求两条已知直线与圆锥曲线的交点"问题，后者的计算要简单得多。

### 算法的四个步骤

**步骤1：用矩阵表示圆锥曲线**

在齐次坐标下，圆锥曲线 \( C \) 可以表示为一个对称的 \( 3 \times 3 \) 矩阵。一个点 \( \mathbf{x} = (x, y, 1)^T \) 在圆锥曲线上当且仅当：
\[
\mathbf{x}^T C \mathbf{x} = 0
\]

**步骤2：构造由已知直线和给定直线生成的退化圆锥曲线**

假设我们有一条已知直线 \( \mathbf{l} \)（也就是要求与圆锥曲线相交的那条直线），再选择另一条任意直线 \( \mathbf{m} \)。这两条直线可以生成一个**退化的二次曲线** \( D \)：
\[
D = \mathbf{l} \mathbf{m}^T + \mathbf{m} \mathbf{l}^T
\]

退化二次曲线在几何上代表两条直线的并集。这类矩阵的秩为2，且行列式为0。

**步骤3：构造直线束中的退化曲线**

关键一步是考虑退化二次曲线的"线性组合"：
\[
C_{\lambda} = C + \lambda D
\]

对于某个特定的参数 \( \lambda \)，矩阵 \( C_{\lambda} \) 会变得**奇异**（即 \( \det(C_{\lambda}) = 0 \)）。这个条件对应于一个关于 \( \lambda \) 的三次方程（在两条圆锥曲线相交的通用情况下）。

解出满足 \( \det(C_{\lambda}) = 0 \) 的 \( \lambda \) 值后，得到的 \( C_{\lambda} \) 是一个**过原圆锥曲线与直线束所有交点的退化二次曲线**。

**步骤4：从退化曲线中提取交点**

由于 \( C_{\lambda} \) 是退化的（秩为1或2），它可以分解为两条直线的乘积。这两条直线与原始直线 \( \mathbf{l} \) 的交点，就是原始圆锥曲线与直线 \( \mathbf{l} \) 的交点。

### 实际应用示例

在R语言的**RConics**包中，就实现了这一算法：

```r
# 定义圆锥曲线（椭圆：半轴 a=8, b=2，倾角 -π/3）
C <- ellipseToConicMatrix(c(8,2), c(0,0), -pi/3)

# 定义直线（齐次坐标表示）
l <- c(0.25, 0.85, -3)

# 计算交点
p_Cl <- intersectConicLine(C, l)

# 可视化验证
plot(ellipse(c(8,2), c(0,0), -pi/3), type="l", asp=1)
addLine(l, col="red")
points(t(p_Cl), pch=20, col="red")
```

该函数的实现正是基于Richter-Gebert书中第11.3节介绍的理论。

### 为什么采用这种矩阵方法而不是直接解方程？

1. **数值稳定性**：矩阵方法在处理退化情况（如直线与圆锥曲线相切）时更加稳定
2. **统一的代数框架**：与书中整体强调的"用代数语言优雅表达几何"的理念一致
3. **易于推广**：该方法可以自然推广到求两条圆锥曲线的交点（书中第11.4节），而不仅仅是直线与圆锥曲线
4. **齐次坐标下的完整性**：能够自然地处理无穷远点（即直线与抛物线平行时的情况）

---

**算法的核心思想**：通过构造一个退化的辅助二次曲线，将原始问题转化为两条直线的简单交点问题。这种几何与代数巧妙结合的方式，正是Richter-Gebert在全书中所倡导的研究范式。

#### 语法形式

::: {.messagebox title="Syntax"}
- `conics/intersect cl={\C,\U,\V,\P,\Q}`: 采用参数化方法求圆锥曲线与直线的交点
  - `\C`: 圆锥曲线的系数矩阵
  - `\U,\V`: 直线上两点的齐次坐标，列向量
  - `\P,\Q`: 交点的齐次坐标，列向量，顺序是按从 $U$ 到 $V$
- `conics/intersect cn={\C,\l,\P,\Q}`: 采用 Richter-Gebert 算法求圆锥曲线与直线的交点
  - `\C`: 圆锥曲线的系数矩阵
  - `\l`: 直线的齐次坐标，列向量
  - `\P,\Q`: 交点的齐次坐标，列向量，顺序是直线的正方向
:::
